# 🛒 쇼핑 리스트 앱

간단하고 직관적인 쇼핑 리스트 관리 웹 애플리케이션입니다.

## 기능

- ✅ 쇼핑 아이템 추가
- ✅ 아이템 완료 체크
- ✅ 아이템 삭제
- ✅ LocalStorage를 통한 데이터 저장
- ✅ 전체/완료 아이템 개수 표시
- ✅ 반응형 디자인

## 사용 방법

1. 입력창에 구매할 아이템을 입력합니다
2. "추가" 버튼을 클릭하거나 Enter 키를 누릅니다
3. 체크박스를 클릭하여 구매 완료한 아이템을 표시합니다
4. "삭제" 버튼으로 불필요한 아이템을 제거합니다

## 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage API
